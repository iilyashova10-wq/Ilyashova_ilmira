# Лабораторная работа №13: Работа с API VirusTotal

## Студент: Ильяшова Ильмира Толеубаевна

### 📋 Описание проекта
Данный проект представляет собой Python-скрипт для взаимодействия с API антивирусной системы VirusTotal.
Скрипт выполняет авторизацию, отправляет запрос и выводит результаты сканирования файла.

### 🎯 Цель работы
- Научиться работать с API антивирусных систем
- Освоить методы авторизации при обращении к API
- Получить навыки обработки JSON-ответов
- Закрепить практические навыки программирования на Python

### 🔧 Функциональность
- ✅ Авторизация по API-ключу (заголовок x-apikey)
- ✅ Запрос информации о файле по SHA256 хэшу
- ✅ Вывод полного JSON-ответа в отформатированном виде
- ✅ Статистики сканирования
- ✅ Детальный вывод результатов проверки
- ✅ Обработка ошибок и исключений

### 📦 Требования
- Python 3.6 или выше
- Библиотека requests
- API-ключ VirusTotal (предоставлен в коде)
